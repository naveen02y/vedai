# VedaAI Frontend

VedaAI is an AI-powered learning platform designed to help students and teachers manage assignments, learning resources, and classroom activities in a smarter and easier way.

This frontend application is built using Next.js and focuses on providing a clean, fast, and responsive user experience.

---

## Features

- AI-powered learning and assignment support
- Modern and responsive dashboard UI
- Mobile-friendly design
- Fast development with hot reload
- Clean navigation and user-friendly layout
- Backend and MongoDB integration

---

## Tech Stack

The project is built using:

- Next.js
- React
- TypeScript
- Tailwind CSS
- Node.js
- MongoDB

---

## Getting Started

### Step 1: Install Dependencies

Open the frontend folder and run:

```bash
npm install
```

### Step 2: Start the Development Server

```bash
npm run dev
```

Once the server starts, open:

```bash
http://localhost:3000
```

in your browser.

## Development

You can edit files inside the `src` folder.

Any changes you make will automatically update in the browser while the development server is running.

---

## Backend Connection

Make sure the backend server is also running for all features to work properly.

Backend server:

```bash
http://localhost:5001
```
